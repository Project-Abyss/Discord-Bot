# Discord-Bot
## Background (專案概述)
根據甲方所提供的七個公告範本，製作出一可套用特定模板並發送公告的 Discord 公告機器人。

主要使用 Python 的 discord.py 套件撰寫機器人的指令，包括：
* 呼叫機器人
* 規範模板格式
* 將使用者輸入的內容套用於模板
* 發布公告
